# Yama Notları V.0.1

Adım Bercan. 19 yaşında 2. sınıf Bilgisayar Programcılığı önlisans öğrencisiyim. Hedeflerim arasında Bilgisayar mühendisliği ve Yazılım mühendisliği gibi alanlar bulunmakta.
